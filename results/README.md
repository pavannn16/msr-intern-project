# Evaluation Results

This directory contains evaluation results from different stages of the exercise.

## Files
- `baseline_results.txt` - Baseline Llama-3.2-1B performance (no quantization)
- `exercise1_results.txt` - Results after quantizing linear layers
- `exercise2_results.txt` - Results after quantizing KV cache (pending)
- `exercise3_results.txt` - Results with E5M3 scale factor (optional, pending)

Note: Some files may be missing until the corresponding exercise is completed.

## Expected Format
Each result file should contain:
- Accuracy metrics
- Runtime/latency measurements
- Memory usage (if applicable)
- Command used for evaluation
- Date and timestamp
